## Retrival Augmented Generation Techniques

This repository contains a collection of **Retrieval-Augmented Generation (RAG) experiments and implementations** exploring different techniques for improving retrieval quality, context construction, and reasoning in LLM-based systems.

The notebooks demonstrate a wide range of RAG strategies, starting from **basic RAG pipelines** and progressing to **advanced architectures such as Self-RAG, Corrective RAG (CRAG), GraphRAG, and Agentic RAG**.

Topics covered include:

* **Basic RAG pipelines** using vector databases and embeddings
* **Chunking strategies** such as semantic chunking, proposition chunking, and contextual headers
* **Query enhancement techniques** like query transformation and hypothetical document embeddings (HyDE)
* **Retrieval optimization** including reranking, fusion retrieval, and adaptive retrieval
* **Context enrichment and compression** for improving LLM input quality
* **Advanced RAG architectures** like Self-RAG, Reliable RAG, and Agentic RAG
* **Hierarchical and graph-based retrieval systems** (RAPTOR, GraphRAG)
* **Multimodal RAG approaches** combining text and visual data
* **Explainable retrieval techniques**

The goal of this repository is to provide **practical implementations and experiments that illustrate modern RAG design patterns**, helping researchers and developers understand how to build more accurate, scalable, and reliable LLM applications.
